TEMA 1: Etiquetas básicas de HTML

El lenguaje HTML nos permite definir los elementos que compondrán una página web mediante el uso de instrucciones. Según Equipo Vértice. (2009), las instrucciones u órdenes que se le dan al navegador se llaman etiquetas. Lo que significa que un archivo HTML está compuesto por muchas etiquetas además de otros elementos.
